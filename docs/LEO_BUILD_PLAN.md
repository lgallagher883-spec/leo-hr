## Platform Components

The LEO platform is composed of the following primary components:

- LEO CORE™ — Decision and reasoning engine
- LEO KNOWLEDGE™ — Organisational knowledge and retrieval engine
- LEO DRAFT™ — Document generation and workflow execution engine
- LEO INSIGHT™ — Analytics and organisational intelligence engine
- Matter System — Case management and audit trail
- Dashboard — User interface and workspace
- API Layer — Communication between the application and the platform services
